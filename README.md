# Basic-Video-Assistant
Basic Video Assistant is a basic AI Video Assistance system that is built using Python. It uses Gemini AI and Gemini API Key to respond to user responces. This is my first project where I have vibe coded it with the help of Claude. 
# Workflow
1. Web cam opens in a tab and the sample video opens in an another tab.
2. The video will be paused untill the user speak out his request.
3. Once the user done with his question, the answer is generated.
4. The answer or the responce will be read and the video starts to play simultaneously.
# Simple example
It is an experimentary idea that what if there exist an interactive video assistant as like a voice assistant.
